CSS Interview QnA

What is the difference between flex row and column?

Flex row and Flex column both are flex direction properties that define how the children element of the flex parent should be alinged. The property is given in the parent element and the flex children inherit them and can be called only when the display is given to be flex. Flex row considers the horizontal axis as the main axis and the vertical axis as the cross axis wheras wheras flex column considers the horizontal axis as the cross axis and the vertical axis as the main axis.

Explain inline , internal and external stylesheet.

Inline , internal and external, all 3 of these are styling techniques. While the end result of all 4 tends to be same, the place where they are mentioned changes their type.

Inline stylesheet: They are the ones that are given next to the tagname in the html element and always holds the highest priority over the other two.

Example:

Inline stylesheet can be given as 
    <div style="color: red;"></div>
Internal stylesheet: They are the ones that are given inside the head tag of the same html file by calling it by the tag style within it.

Example:

Internal stylesheet can be given as 
    <head>
    <style>
    color: red;
    </style>
    </head>
External stylesheet: They are the ones that are given in another styling sheet and are later linked to the current html page by using the link tah and they are the ones that are most preferred as they make the work look neat and easily understandable.

Example:

External stylesheet can be called as 
    <head>
    <link rel="stylesheet" href="style.css">
    </head>

Justify-content allows you to do what?

Justify-content is a property in CSS that can be given to the parent flex or grid container which alings the children element along the horizontal axis based on the value given to the Justify-content property. Justify-content can take the following values:
  • flex-start
  • flex-end
  • center
  • space-between
  • space-around
  • space-evenly

What is the difference between absolute and relative positioning?

Absolute and relative positioning, both are the position related properties. While absolute positioning ensures that the element is relatively positioned to its first non-static ancestor or its parent element, relative position based on it's current position and acts upon the baseline value hence the moves in a reverse direction of top, bottom, left and right given to it.

What is grid-template-columns used for?

Grid-template-columns is a CSS property and comes under the grid that is used to assign the number of columns that we require. By default we give a size of auto that ends up taking the length of entire webpage and aligns equal space to all the columns if nothing overflows, and if it does, it automatically adjusts. We can even give our own custom width other than auto.

What is the z-index in CSS?

The z-index is a 3D property in CSS that specifies the stack order of an element. By default every element has an z-index value of 0, but depending on our requirement, we can change it either positive values to bring it on top or negative values to send it to back. 
    <div class1 style="z-index: -1;"></div>
    <div class2 style="z-index: 0;"></div>
    <div class3 style="z-index: 5;"></div>

What is the difference between padding and margin?

Padding and Margin both are CSS box model properties. While Padding is the free space around the content area that takes up the colour of the background color of the element, margin is the distance that we want for the specified element from another element and it has no background color of it's own.

What is box sizing?

Box-sizing is a CSS property that determines whether padding and border should be included in an element's width and height or not. It takes two values

Box-sizing: content-box;
It's the default value of box-sizing. In here the content height and width are that of the element.

Box-sizing: border-box;
In here the element's height and width include that of element along with it's padding and border.

What is animation delay?

Animation delay is a CSS animation property that is used to hold the animation for a specified time hence delaying it to happen. It takes the value of time in milliseconds and seconds.

Which property you will use to merge cells vertically in grid?

We use "grid-row: from/(to-1)" property to merge cells vertically and it has to be given to the targrted cell.

Which property you will use to merge cells horizontally in grid?

We use "grid-column : from/(to-1)" property to merge cells horizontally and it has to be given to the targrted cell.

Explain CSS box model

CSS box model assumes every element in a HTML page as an rectangular box and can be found through the inspect button on right clicking on a webpage. It consists of content, padding, border and margin from within to outside respectively.

what is the difference between display none and visibility hidden?

Display : none and Visibility : hidden both are used to hide an element from the webpage but the only difference is that display : none doesn't retain any space of the hidden element and will be occupied by another element whereas visibility : hidden retains the space of the hidden element on the output page.
