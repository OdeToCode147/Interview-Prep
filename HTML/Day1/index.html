HTML Interview QnA

What is the difference between div and span?

Div tag is abbreviation for Division tag which is block level in nature, which means it is bound to take full length of the width of the screen irrespective of it's content and hence always starts in a new line and is often used to wrap the section of the document. Whereas Span happens to be an inline element that takes only the width that it's content requires and if it happens to come next to another inline tag, it starts within the same line and hence are used to wrap smaller portions of text or images within the document.

What is the difference between class and IDs?

Both the Class and ID are used to identify the elements within the html file, but the difference is that the ID is unique to every element and one element can have only a single id, whereas class can be used to identify multiple elements and even a single element can have multiple classes. But the ID has an highest priority over Class when both of the attributes have a conflicting value.

Example:

ID can be written as 
    <div id="ID"></div>
Class can be written as 
                <div class="Class1 Class2"></div>

What are self closing tags?

Self closing tags are the type of tags that doesn't have a separate closing tag as they get closed within the opening tags itself. These tags are also known as empty tags since the element only contains the tag and nothing else like the attributes.

Example: 
                <br> , <hr> , <img> , <input> , <area>

What is the difference between block and inline?

Block level elements by default is bound to take full length of the width of the screen irrespective of it's content and hence always starts in a new line and by default has an added margin.

Example: 
                <div> , <h1> , <p> , <pre> , <header> , <footer> , <section> , <form> , <nav> , <hr> , <blockquote> , <li>
Whereas an inline element takes only the width that it's content requires and if it happens to come next to another inline tag, it starts within the same line and hence they don't have an added margin and even styling like padding and margin doesn't get applied on it.

Example: 
                <a> , <b> , <i> , <span> , <button> , <lable> , <sub> , <sup> , <select> , <object> , <img> , <em> , <strong>
But the nature of an element being block or inline can be easily changed in CSS by changing their display type.

What is the difference between em and strong?

em and strong both are special tags that can used as an alternative to i and b tags respectively. While em tag appears in italic and is used to emphasize the text that makes it stand out in the crowd, the strong tag that appears in bold is often used to convey the importance and urgency of the text.

What are attributes?

Attributes are special tag specific HTML keywords that provides an additional information for that HTML element. They come in the form of name and pair values that must be written in the opening tag of the element anywhere after the tagname.

Example:

<a> : href, rel, target, type

<img> : src, alt, width, height

<ol > : start, type, reversed

Explain br tags.

br tags are used to create a line break within a text or sections. They are empty tags, i.e., they don't need a separate closing tag and are inline in display nature. Though they aren't much preferred and are often replaced by using the div or span elements.

What are semantic tags?

Semantic tags are special kind of tags that are self defining unlike the other tags that the Browser or User can look at it's tagname and easily identify it's function within the HTML webpage.

Example: 
                <header> , <footer> , <section> , <form> , <table>
